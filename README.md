# ZyraWear
ZyraWear is a modern and stylish ecommerce platform focused on clothing and fashion. It provides users with a seamless shopping experience, enabling browsing, filtering, and purchasing of apparel with ease. The platform supports user authentication, product management, shopping cart functionality, order tracking, and payment integration to deliver a smooth online shopping journey.

Features
✅ User registration and login with role-based access (customers and admins)

✅ Browse and filter clothing products by category, price, size, and color

✅ Detailed product pages with images, descriptions, and reviews

✅ Add to cart and update cart quantities

✅ Secure checkout with integrated payment gateway (e.g., Razorpay or Stripe)

✅ Order history and status tracking for customers

✅ Admin panel for managing products, inventory, and orders

✅ Responsive UI designed with React and Tailwind CSS

✅ Backend REST API built with Node.js and Express

✅ MongoDB used for data storage

Technology Stack
Frontend	Backend	Database	Payment Gateway
React, Tailwind CSS	Node.js, Express	MongoDB	Razorpay / Stripe
